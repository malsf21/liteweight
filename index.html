Liteweight

An educational CSS framework by Matthew Wang and Nick O'Brien.

Colours

An important thing to know about liteweight is that most of our elements rely on our colour map. Here's what the default is:

Primary (#E95420)

Secondary (#7AD0ED)

Success (#2BD937)

Danger (#F01A1A)

Caution (#D9D92B)

Info Colour (#6878ED)

These colours can be edited in liteweight/variables.scss; they are often edited by themes.

Typography

Here's all the typographical elements that Liteweight modifies.

Headings

Liteweight doesn't modify the <h-*> tags directly; instead, it uses modifier classes to allow your code to be both semantically correct and rockin' awesome.

Heading 3

Text

Liteweight also doesn't directly modify <p>; instead, it uses the class="text" to style elements. You can use it on a variety of elements.

The text class is a foundation for more modifier classes. You can change the colour with text-* (* referring to all the available colours in $colours)

Cards

Cards are a nice class that wraps anything that's inside them. They can contain anything and everything.

This is a card (with just padding). 
			
<div class="card">
	This is a card (with just padding).
</div>
This is a default card (with a background). 
			
<div class="card card-full">
	This is a default card (with a background).
</div>
This is a card with the "primary" background. You can use any colour in our colour array for card backgrounds. 
			
<div class="card card-bg-primary">
	This is a card with the "primary" background. You can use any colour in our colour array for card backgrounds.
</div>
This is a card with the "primary" border. You can use any colour in our colour array for card borders. 
			
<div class="card card-border-primary">
	This is a card with the "primary" border. You can use any colour in our colour array for card borders.
</div>

Images

block-responsive is a simple class: it makes any block element (like an image) expand to its parent width or its original width, whichever comes first.

			
<img class="block-responsive" src="#">

block-balance does something similar, except it also sets margin-left and margin-right to auto

			
<img class="block-balance" src="#">
